# 语言服务器

### 是什么

语言服务器是 vscode 的 **一种特殊扩展**，为多种编程语言提供强大的编辑器体验。

### 能干什么

借助语言服务器，能够实现**自动完成、错误检查、跳转到定义等多种vscode支持的其他语言特性**

# LSP协议

### 解决了什么问题

- 规范语言服务与编辑器之间通信 ———— 语言服务使用其原生编程语言与具有Node.js环境的VSCode集成困难，同时不同编辑器涉及不同环境。
- 独立进程运行，避免性能开销 ———— 语言特性通常涉及大量资源开销。
  例如：文件解析，构建语法树，执行静态分析
